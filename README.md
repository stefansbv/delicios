Delicios
========
Ștefan Suciu
2014-11-28

Version: 0.10

A cooking recipes web application built with Dancer2 and Bootstrap3.

IMPORTANT: Archived, read only repository!

Requirements
------------

Perl and Dancer2.


The Database
------------

Generate the database schema:

```bash
$ dbicdump -o dump_directory=./lib schema.conf
```
